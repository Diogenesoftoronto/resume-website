There are over 250+ npm packages of themes that we pick official themes from.

Want to develop your own?

Before you publish your theme to npm, you can work on it locally by putting the files beside your resume.json file. When you run `resume-cli serve`, it will look for a local index.js first.

Our theming system was released just recently and new themes have already started popping up. If you would like to develop a theme for the community please visit the theme-manager repo.

Getting started

If you are using the registry to host your resume, you can easily test different themes by appending a query string e.g.

http://registry.jsonresume.org/thomasdavis?theme=flat

Or you can set the default theme for your resume on the registry by using the --theme option in the CLI tool e.g.

resume publish --theme flat
